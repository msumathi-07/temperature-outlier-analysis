# Temperature Outlier Detection Project

## Description
This project detects abnormal spikes in temperature sensor data using Z-score and smooths them using rolling average.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib

## Key Steps
1. Data cleaning
2. Outlier detection (Z-score)
3. Smoothing (Rolling average)
4. Visualization

## Output
- Cleaned dataset
- Graph showing original vs smoothed data
